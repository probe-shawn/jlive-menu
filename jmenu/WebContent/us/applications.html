Applications

jLIVE applications module include databases, forms, reports and executable programs. After pressed the [New Create Application] button, The main control panel will be appended each module nodes. Before create new application, must boot the jLIVE middle ware system first and fill in the middle ware URL where the connection will be establised. Developer can press the [test connection] button to test the connection status. 
New create application :  Create a new application.
Test connection :  To check the middle was booted or not.
Cancel :  Cancel
Save update :  Save the modified.
Delete :  Delete the application, system just only remove the control panel node. The associated data files still existed.
Prj. Name :  jLIVE system can generate multiple executable programs by one project name.
Http :  jLIVE middle ware URL
Login :  Prompt the login dialog box, when user log in the application. This module must deploy the jLIVE security control. 
Author :  Author.
Version :  Version.
Path :  The project default path.
Access Key :  The access resources authority key for work group version only.
Work group :  According to the authority licenses , the workgroup client can scan and run the same project which developed by other programmers. Fill the computer name or IP which the one in the team work, fill the access key which the one specified. 
Description :  The description or note for the application.
