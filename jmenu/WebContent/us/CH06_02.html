6.2 Report Management

　

在jLIVE Builder 的設計畫面(程式控制面版)中，報表系統乃是在視窗畫面底下的一個分類節點，使用者可以在程式控制面版的工具列，報表名稱、路徑設定頁上的工具列或在該報表名稱、圖示上按滑鼠的右鍵來呼叫出功能選單，並可執行編輯、存檔、刪除、編譯等等動作。

　

  • Design : Go into design panel.

  • Make Java : Make java source code.

  • Compile Java : Compile java source code.

  • Build Java Class : Build java source code into class.
