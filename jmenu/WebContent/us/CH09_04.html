9.4.1 Common Properties

　

General properties :

  • Event name

  • Condition : it have to satisfy the condition before execute event

  • Otherwise : you can choose another event if not satisfy the condition

  • Document note

  • Next event : After execute the event, you choose next event to enforce

　

▲Top

　

9.4.2 Copy DataSource Data To Buffer DataSource

　

To use this event, you have to create a buffer datasource first.

  • DataSource : choose one datasource

  • Buffer DataSource : choose buffer datasource

  • Append Record Data

  • Transfer Filter : it can transfer field data

　

▲Top

　

9.4.3 Copy Server Text File To Buffer DataSource

　

This event is able to copy text file into buffer datasource.

  • File Url : file position in server

  • Tab Delimited

  • Define Text Record : identify format and suitable buffer datasource field

  • Buffer DataSource : choose one datasource

  • Map Transfer Fields : the text field format map to buffer datasource

9.4.4 Upload File

　

You can upload datasource into file.

  • File Url : file position

  • Append

  • Initial Event

  • Content : upload data

  • Break Cndition True will break loop

  •  next event
