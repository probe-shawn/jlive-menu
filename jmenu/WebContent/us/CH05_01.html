Form is the operate interface of application. One application usually include lots of forms. So form is the basic unit in windows operation system. Not alike windows, JAVA defines the basic unit in panel. If the application executes in jar file, it acts very similar to form. Supposing that the application executes in html mode, the panel still is a part of web page. In order to realize easily, we named the panel as form. When we design the application, it should be a fine way to map out it in form.
