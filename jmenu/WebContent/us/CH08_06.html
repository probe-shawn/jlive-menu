8.6.1 DS

  • Datasource : Choose one datasource.

  • Datafield : Choose one field inside datasource.

  • Recheck when any component value changed.

  • Button column ? column : Depend on your need.

  • Radio Button : You can create new button, delete, post, up and down those buttons.

  • Text : Make a description of the button.

  • Mnemonic : You can choose this checkbox without click mouse, only press " ALT " and your define key at the same time.

  • Text Horizontal Position : It is used to adjust horizontal position.

  • Text Vertical Position : It is used to adjust vertical position.

  • Paint Options :

    • Paint Focus : Focus the component when you press it.

    • Paint Fill : The component will fill other color when you press it.

    • Paint Border : Look like a button and could be press down.

  • Display Icon : Choose one icon to display.

  • Press Icon : Choose one icon, only show it when press the checkbox.

  • Roll Over Icon : Choose one icon, show it when the mouse roll over it.

  • Selected : Condition formula

  • Data : Data for select

  • Font : Choose one kind of font in your computer.

  • Size : Choose the font size.

  • Style : Check one style. Bold and italic.

  • Color : Choose foreground color and background color.

  • Component Tool tips : When your mouse move on the component, it will show the string.

8.6.2 Event

  • Action : When you click " Enter " button.

  • Gain Focus : While the mouse click into the component.

  • Double Click : While double click the right button of mouse.

  • Right Click : While click right button of mouse.
