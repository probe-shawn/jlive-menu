9.3.1 Common Properties

　

General properties :

  • Event name

  • Condition : it have to satisfy the condition before execute event

  • Otherwise : you can choose another event if not satisfy the condition

  • Document note

  • Next event : After execute the event, you choose next event to enforce

　

▲Top

　

9.3.2 Open Form

　

This event can open another form.

  • Form : choose one form

　

▲Top

　

9.3.3 Show MessageBox

　

Show message box :

  • Caption : set title

  • Message : Output message

  • Message Type :

    • Error

    • Information

    • Warning

    • Question

    • lain

  • Options Type :

    • OK

    • Yes No

    • Yes No Cancel

    • Ok Cancel

  • Yes Ok event : when choose this button, it will active another event

  • No event : when choose this button, it will active another event

  • Cancel event : when choose this button, it will active another event

　

▲Top

　

9.3.4 Execute Print Job

　

It will execute the print job.

  • Reports : choose one report

9.3.5 ExecuteFormula
