Formula Design Pane

The Formula is a high performance interpreter expression, support for computing data and control program process flow. there are over 100 functions divided into Math, Compare, Logic, String, Date, Function, Components, Variable, DsOperation, Flow and  Special categories. 

  • The Data Source Fields, components, System variables and Globe and Local Hashtable can be used as formula operand.

  • The Data Source meta-information such as Record Number, EOF, BOF, IsEmpty, Insertion Mode, Update Mode, Total Record and Read Only can be used within formula computing.

  • The formula and functions will return a string value always, that could be a computing result or a process function return state. If that be used as flow condition, the empty string or "0" value mean to FLASE state, otherwise was TRUE.

  • The IF-THEN-ELSE is nested expression.

  • The parameter of Formula function, could be another expression.
