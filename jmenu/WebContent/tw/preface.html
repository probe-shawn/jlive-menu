問題與建議

　

　

感謝您採用 jLIVE Builder™ 作為網際網路 Java™ 應用程式開發的平台。

　

jLIVE Builder™  提供了最先進的視覺化開發環境，以軟體元件技術 (component technologies) 讓設計師快速的開發網際網路企業應用程式。系統中內設的多層式架構 (multi-tiers) 不但是目前企業所需，Java™ 技術的導入更整合了企業跨平台的作業需求。從系統設計、流程、除錯、報表到系統導入，jLIVE Builder™ 完整的解決方案，讓開發者能隨時因應多變的商務需求，快速地開發、整合企業應用程式。

　

　

本書的結構


本書分為15章 (chapters)，及若干附錄文件，各章節的內容與重點概述如下表。
　

　 第 1章 簡介
             (Introduction)

jLIVE Builder™  的安裝與啟動、系統特性、JDBC 驅動程式、系統說明等。
　 第 2章 技術機能概論
             (Technical overview)

網際網路的企業應用整合 (Enterprise Application Integration) 、多層式架構 (N-tier architecture) 下網際網路的考量 (Internet thinking) 、資料流量與伺服器負載 (Data transferring and server loading) 、授權式安全控管 (Authorized security control) 、網際網路報表作業 (Internet report system) 等等。
　 第 3章 專案開發程序
             (Project develop procedure)

應用程式系統開發程序 (Application develop procedure) 、jLIVE Builder™ 程式設計概念 (Programming concept) 、視窗設計 (Form design) 、報表設計 (Report design) 等。
　 第 4章 應用程式
             (Application)

新增應用程式 (Application) 時，各個屬性的意義與設定、群組 (Work  Group) 的設定、應用程式的輸出 (Export) 、輸入 (Import) 、應用程式線上更新 (Online Update) 等等。
　 第 5章 資料庫
             (Database)

新增資料庫 (Database) 時，各個屬性的意義與設定。
　 第 6章 視窗
             (Form)

新增視窗 (Form) 時，各個屬性的意義與設定、視窗屬性 (Form  Properties) 、視窗管理 (Form Arrangement) 、視窗設計平台操作 (Design  Framework) 、除錯與中斷事件 (Debug and Event Interrupt) 。
　 第 7章 報表
             (Report)

列印流程機制 (Printing flow and mechanical) 、新增報表 (Report) 時，各個屬性的意義與設定、報表屬性 (Report properties) 、報表管理 (Report  arrangement) 、報表設計平台操作 (Design framework) 、除錯與中斷事件 (Debug and event interrupt) 、報表元件 (Report printable components) 。
　 第 8章 執行程式
             (Execute program)

瀏覽器執行程式 (Browser enable applet run program) 、獨立執行程式 ( Standalone [Jar run] program) 、授權式安全控管設定 (Authorized security  control) 。
　 第 9章 資料來源
             (Data Source)		 資料來源設計規劃程序 (Data source design procedures) 、資料來源的類別 (Database type) 、資料庫資料表列示 (Database tables listing) 、資料來源列示 (Data source listing) 、全域資料來源 (Globe data source) 、資料來源讀取、新增、存檔、刪除的系統內部程序 (Fetch, insert, post, delete system  internal procedures) 。
　 第10章 元件
              (Component)

元件類別 (Components) 、配置 (Layout components) 、使用和調整 (Use and adjust components) 、屬性設定 (Common properties) 、觸發事件。
　 第11章 事件
              (Event)

事件類別 (Event type) 、事件列表 (Event table) 、事件流程綱要 (Event flow schema) 。
　 第12章 運算公式
              (Formula)

運算公式類別 (Formula type) 、運算子及函式 (Operator and function) 、運算公式編輯視窗 (Formula design window) 、運算公式設計 (Formula design) 、運算公式資料型別 (Formula data type) 、運算資料的取值 (Retrieval computed data) 、變數 (Variable) 、運算子 (Operator) 、函式的參數 (Parameters of function) 、運算公式結果值 (Formula result data) 、IF-THEN-ELSE 流程控制 (IF-THEN-ELSE flow control) 。
　 第13章 國際化
              (Internationalization)		 使用者程式介面國際化 (I18N user interface) 、資料儲存及傳輸國際化 (I18N data storage and transfer) 、分散式動態語系登轉 (Distributed  language mapping) 、支援的語系 (Select supported language) 、視窗及報表語系對照 (Form and report language mapping) 、資料庫語系與國際字元欄位 (Database language supporting and national character) 、使用者語系切換介面 (User language interface) 、編譯及佈建程式 (Compile and deployment) 。
　 第14章 應用程式技術文件
              (Technical Document)		 執行技術文件產生系統 (Execute jLIVE Document Builder™) 、技術文件首頁屬性設定 (Technical document home page properties) 、技術文件視窗屬性設定 (Technical document form properties) 、技術文件選項設定 (Technical document options properties) 、技術文件文件格式檔 (jLIVE Document Builder™ document pattern) 。
　 第15章 應用程式伺服器
              (Application Server)		 jLIVE™ middle ware (jLIVE™ middle ware) 、jLIVE™ middle ware 管理介面 (jLIVE™ middle ware management interface) 、異地應用伺服器同時存取 (Multiple application servers access simultaneously) 、jLIVE™ middle ware 安裝 (jLIVE™ middle ware installation) 。
