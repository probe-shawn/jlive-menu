執行超連結 (execute hyper link)

　

事件執行狀態與訊息 (event runtime status and message)

　

執行超連結的功能，內定為開啟一個新的瀏 覽器視窗。

  • 超連結

    　

    • Ref ： 相對 Relative (class/doc/)。如果核取 (checked) ，則超連結的檔案，必須存在於應用程式下的class/doc/ 目錄中。取消核取 (unchecked)，則需以運算式組合其絕對位址(URL)。

    • 編碼 (encode)： 參數使用的編碼字元集 (character Set) 如 ： UTF-8，MS950，MS932 ...。

    • 參數 (parameters)： 如果這個超連結需要參數的話，可以定義多組傳入的參數名稱 (name) 與參數值 (value)。

    • 視窗標題 (window title)： 定義超連結的瀏覽器視窗標題。

事件執行狀態與訊息 (event runtime status and message)

　

EvStatus("事件名稱") ：取得事件執行狀態。
　

回傳值 狀態
空字串 事件未被引發。
-1 事件啟動後，依條件判斷後執行否則事件。
1 事件啟動後，執行成功。
0 事件啟動後，執行過程有誤。

　

EvStatus("事件名稱.error") ：取得事件執行錯誤代碼。

　

回傳值 說明
空字串 或 0 沒有錯誤訊息或無法取得錯誤訊息。
1 執行超連結錯誤。
4 不支援指定的編碼 (unsupported encoding )。

　

EvStatus("事件名稱.INFO_STRING") ：取得事件執行 INFO_STRING 的訊息。

　

INFO_STRING 回傳值 說明
name URL 超連結URL。
