日期運算式是針對日期做處理。日期的原型格式 (original type)為 "yyyyMMdd"，時間的原型格式為 "HHmmss"，日期時間的原型格式為 "yyyyMMddHHmmss"。設計師以常數給值或由元件 (屬性格式為日期時間格式樣版)、資料欄位 (欄位型態為日期時間) 取值時，傳回的值皆為原型格式。

　

　

　

Today()

　

說明： 客戶端(Client)的系統日期。
參數： 無。
傳回值： 客戶端(Client)的系統日期，為 8 byte 字串。
範例：

假設系統日期為 2000/11/09      傳回      "20001109"

FormatDateTime(pattern,datetime)

FormatDateTime(pattern,datetime, locale)

　

說明： 將 "datetime" 以自訂的日期時間格式重新排列。
格式代表符號如下。
參數： "datetime" 基準日期時間。
"pattern" 自訂日期時間的格式，參閱 日期格式化符號。
"locale" 預設不輸入時為西元，"zh_TW" 為台灣民國，"ja" 為日本和曆。
傳回值： 日期時間。
範例：

FormatDateTime("yyyy-MM-dd HH:mm:ss" , "20001109160138" )   

     傳回  "2000-11-09 16:01:38"

　

FormatDateTime("MMM-dd-yyyy" , "20001109160138" )

     在中文語系OS下傳回  "十一月-11-2000"；

     英文OS則傳回  "Nov-11-2000"


FormatDateTime("MMMM-dd-yyyy" , "20001109160138" )

     在中文語系OS下傳回  "十一月-11-2000"；

     英文OS則傳回  "November-11-2000"


FormatDateTime("hh:mm:ss" , "20001109160138" )   

     傳回  "04:01:38"
　

FormatDateTime("yyy年MM月dd日", "20001109160138", "zh_TW" )

     傳回  "90年11月09日"

FormatDateTime("yyy年MM月dd日", "20001109160138", "ja" )

     傳回  "平成13年11月09日"
　

套用各種符號所傳回的日期時間格式請自行測試。

NextDateTime(datetime,second)

　

說明： 計算 "datetime"  加上 "second" 之後的日期時間。
參數： "datetime" 基準日期時間、 "second" 秒。
傳回值： 日期時間。
範例：

假設系統日期時間為 "2000/11/10-18:43:36"
NextDateTime( Today() ,86400)                 傳回         "2000/11/11-18:43:36"

ApServerDateTime()

　

說明： 取得Application server (jLIVE middle ware) 目前的日期時間。系統在第一次取得 Application server 的日期時間後，為避免頻繁的查詢Application server，隨即由使用者端以此基準計算時間。設計師可以呼叫 ApServerDateTime(refresh)，重新查詢日期時間。
參數： 無。
傳回值： 日期時間原型 (yyyyMMddHHmmss)。
範例：

假設系統日期時間為 "2000/11/10-18:43:36"
ApServerDateTime()                 傳回         "20001110184336"

設計師可能運用此函數所得的日期時間，以 SetProp("timer", "time", ApServerDateTime()) 方式將 Application Server 的時間設定給計時器元件(timer component)。
