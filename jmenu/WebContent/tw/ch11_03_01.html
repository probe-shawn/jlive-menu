Event

　

資料來源之導覽元件 (data source navigator)

　

事件執行狀態與訊息 (event runtime status and message)

　

事件的執行一如資料來源的導覽元件 (Navigator component)。

在程式實作時，亦可使用功能相同的運算公式：DsNavigator(ds,op_str)，取代此事件。

  • 資料來源 (data source)： 選擇資料來源。
    　

  • 導覽元件 (navigator operation)： 資料來源導覽元件操作功能。

    • 首筆 (first)： 移至資料來源首筆記錄。

    • 上筆 (prior)： 移至資料來源上一筆記錄。

    • 下筆 (next)： 移至資料來源下一筆記錄。

    • 尾筆 (last)： 移至資料來源尾筆記錄。

    • 新增 (insert)： 新增資料記錄。

    • 刪除 (delete)： 刪除資料記錄，預設的刪除動作沒有警告。

    • 編輯 (edit)： 編輯資料記錄。

    • 存檔 (post)： 儲存資料記錄。

    • 取消 (cancel)： 取消編輯。

    • 更新 (refresh)： 重新由緩衝區 (buffer) 的檔案記錄 (record) 讀取，並將連結至此資料來源的相關元件內含值更新。

    • 重開 (reopen)： 重開資料來源。

    • 全刪 (delete all)： 將資料來源的資料全部刪除，此功能只能夠使用於 Buffer 資料來源，若要刪除實體資料來源的全部資料，需使用 NON_SELECT SQL Commands。

    • 查詢 (query)： 查詢資料來源，使用系統內建的查詢模組。

　

　

事件執行狀態與訊息 (event runtime status and message)

　

EvStatus("事件名稱") ：取得事件執行狀態。
　

回傳值 狀態
空字串 事件未被引發。
-1 事件啟動後，依條件判斷後執行否則事件。
1 事件啟動後，執行成功。
0 事件啟動後，執行過程有誤。

　

EvStatus("事件名稱.error") ：取得事件執行錯誤代碼。

　

回傳值 說明
空字串 或 0 沒有錯誤訊息或無法取得錯誤訊息。
1 資料來源名稱不存在。以 EvStatus("事件名稱.name") 取得其名稱。
2 資料來源操作有誤。

　

EvStatus("事件名稱.INFO_STRING") ：取得事件執行 INFO_STRING 的訊息。

　

INFO_STRING 回傳值 說明
name 資料來源名稱 不存在的資料來源名稱。
