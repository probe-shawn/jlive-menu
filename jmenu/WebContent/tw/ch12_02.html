12.2 比較 (compare)

　

x = y

x < y

x > y

x <= y

x >= y

x <> y

　

比較兩個運算元的大小關係，傳回 0 表示偽，傳回 1 表示真。當 x 與 y 的結果值，兩者都可以解析為數值 (numeric) 時，直譯器 (interpreter) 以數值比較大小，否則以字串型式比較大小。

  • 日期函數傳回值為字串 (日期原型格式 yyyyMMdd)。如 Today() = "20001110" 。

  • 數值字串可視同數值(如 "123"=123)，但只要數值字串含有空格，則視為文字字串，如" 1 2 3 " = 0 (左邊空2格，字與字中間空1格，右邊空1格)。

  • 在比較運算式中各種運算子的大小順序：中文字>小寫英文字>大寫英文字>（數值 = 數值字串 =日期函數）>日期字串>空格字串>空字串 。

x = y

　

說明： 檢查 x 運算式是否等於 y 運算式。
參數： "x"、"y"，可為常數值或運算式。
傳回值： 1 表示等於，0 表示不等於。
範例：

2001="2001"         傳回    1
"A"="a"                 傳回    0
