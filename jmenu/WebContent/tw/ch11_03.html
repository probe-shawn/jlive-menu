11.2 資料來源事件 (data source event)

共同屬性 (common properties)

資料來源之導覽 (data source navigator)

設定資料來源紀錄之欄位內含值 (set data source record field data)

觸發資料來源 (trigger data source)

查詢資料來源 (query data source)

執行 NON_SELECT Sql 命令 (execute NON_SELECT SQL commands)

執行預儲程序 (execute stored procedure)

設定資料庫 Transaction Isolation Level (set database transaction isolation level)

設定資料庫 Transaction AutoCommit (set database transaction auto commit)

設定資料庫 Transaction Commit or Rollback (set database transaction commit or rollback)

Transaction 附錄

　

　

　

資料來源事件 (data source event) 在應用程式執行期 (runtime)，執行：

  • 操作資料來源(導 覽、欄位值、觸發、查詢)。

  • 執行 SQL 指令 (SQL、預儲程序)。

  • 設定資料庫(Isolation level、Transaction)
