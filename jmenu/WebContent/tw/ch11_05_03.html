Buffer 資料來源排序 (buffer data source sorting)

　

事件執行狀態與訊息 (event runtime status and message)

　

　

利用此事件可以將 Buffer 資料來源中的資料作排序，優先順序是由上而下。

  • Buffer 資料來源 (buffer data source)： 選擇 Buffer 資料來源。
    　

  • 排序 (order by)： 欄位排序。

    • 向上 ： 將標示欄位上移。

    • 向下 ： 將標示欄位下移。
      　

    • 選擇 (select)： 核取欲排序欄位。
      　

    • 資料欄位名稱 (field name)： 欄位名稱。
      　

    • 降冪 (DESC)： 核取 (checked) 表降冪，不核取 (unchecked) 表升冪，預設為升冪。

事件執行狀態與訊息 (event runtime status and message)

　

EvStatus("事件名稱") ：取得事件執行狀態。
　

回傳值 狀態
空字串 事件未被引發。
-1 事件啟動後，依條件判斷後執行否則事件。
1 事件啟動後，執行成功。
0 事件啟動後，執行過程有誤。

　

EvStatus("事件名稱.error") ：取得事件執行錯誤代碼。

　

回傳值 說明
空字串 或 0 沒有錯誤訊息或無法取得錯誤訊息。
1 Buffer 資料來源名稱錯誤。
2 Buffer 資料來源排序作業錯誤。

　

EvStatus("事件名稱.INFO_STRING") ：取得事件執行 INFO_STRING 的訊息。

　

INFO_STRING 回傳值 說明
name 資料來源名稱 Buffer 資料來源名稱。
