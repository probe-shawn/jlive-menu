圖片引發器元件 (Image invoker component) 是以圖片的不同範圍啟動事件 (fire event)。當使用者以滑鼠移入範圍時，設計師可以設定各個範圍的顏色變化 (由顏色1 至 顏色2 )，及提示文字 (tips)。使用者按下滑鼠 (clicked)時即啟動事件。如果採用jLIVE™ 授權機制，作用範圍連結的事件為開啟視窗事件 (open form) 、執行印表作業事件 (report job) 、查詢資料來源事件 (query data source) 時，皆會根據使用者是否擁有相關授權項目 (IUDQP)，而使該區域致能 (enabled) 或失效 (disabled)。

　

　

　

引發器 (Invoker properties)

  • 背景圖片 (background)：選擇背景圖片。
    　

  • 引發器 (invoker)：設定引發器。

    • 新增引發器 (create new invoker map) ：建立引發器。

    • 刪除引發器 (delete invoker map) ：刪除引發器 。

    • 編輯引發器 (update invoker map) ：編輯引發器 。

    • 上移 (move up) ：次序往前移。

    • 下移 (move down) ：次序往後移 。
      　

    • 範圍 (region) ：設定引發範圍。(左，上，寬，高)。
      　

    • 事件 (event) ：選擇啟動事件。
      　

    • 顏色1 (color 1)： 滑鼠移到引發範圍內時，引發範圍內所有顏色1 (color 1) 將轉變為顏色2 (color 2)。
      　

    • 顏色2 (color 2)： 滑鼠移到引發範圍內時，引發範圍內所有顏色1 (color 1) 將轉變為顏色2 (color 2)。
      　

    • 框線 (border)： 引發範圍框線。

      　

      • 無框線 (no border)：無外框。

      • 顏色框線 (colored border)：引發範圍以顏色2 (color 2) 繪製框線。

      • 按鈕框線 (button border)：引發範圍以按鈕型式繪製外框。
        　

    • 提示 (tips)：滑鼠移到引發範圍時，顯示的提示文字內容。

　

　

　

　

圖片對映區域設定 (invoker region setting)

　

新增一個引發器或以滑鼠雙擊範圍的儲存格， 會開啟圖片對映區域設定視窗。設計師可以滑鼠拖曳，設定引發器的作用範圍，同時工作視窗會顯示作用區域的大小及滑鼠所在位置的顏色值。

  • 工作畫面解說：

    　

    • 作用區域：引發範圍 (左，上，寬，高)。
      　

    • 滑鼠的位置：(X，Y)。
      　

    • 滑鼠所在位置的顏色值：(R，G，B)。
      　

    • 粉紅色的框點：標示作用區域。
      　

    • 調整作用區域大小：以滑鼠拖拉引發器範圍的粉紅色框點。
      　

    • 移動作用區域：滑鼠按住框點內移動。
      　

    　

  • 範例 (example) ：
    　

    • 以滑鼠圈選涵蓋 Customers 字樣的區域；移動滑鼠到藍色的字；檢視(R, G, B)的值為(0, 0, 255)；按下 確定；回到屬性設定頁。

    • 在引發器中選擇一個觸發事件 (event )。

    • 點選顏色1 (color 1) 設定其 RGB 值為 (0, 0, 255) 。
      　

    　

    • 點選顏色2 (color 2) 設定當滑鼠移至時的顏色值為另一顏色 RGB ( 153, 204, 255 ) 。

    • 應用程式執行時，當滑鼠移至作用區域時，Customers 的藍色 RGB(0, 0, 255) 字樣，會變為水藍色 RGB( 153, 204, 255) ，滑鼠在作用區按下時 (clicked)，即會引發設定的事件 (event )。
