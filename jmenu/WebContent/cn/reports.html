Report
jLIVE report system is highly integrated with Form design framework, it support the same visual RAD environment as Form design, drag and drop printable object to layout report. The printable content is dynamic computing by jLIVE formula, so all the components content and data source can be printed. 
Create New Report :  Create a new report.
Cancel :  Cancel
Save :  Save the modified properties.
Delete :  Remove this report.
Make Java :  From XML to generate Java source code.
View Java :  Java source code viewer.
Compile Java :  Compile Java source to class.
Build :  From XML to generate Java source code then compile it to class.
Class Name :  The Report name, cannot be blank, and must follow the Java variable naming convention to name it. System will generate Java class base on this name, and the application will use the name as reference of this report. Change the existed report name, system will check name duplicated or not, then alter all the files which depend on it.
Author :  Author
Version :  Version
XML File :  The report properties xml file. System will generate a new one when first time enter the report design mode.
Java Source :  The generated Java source code.
Class Path :  The default path for Java compiler to output class file.
Description :  Description
