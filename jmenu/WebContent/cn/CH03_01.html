¡@

¡@

To define a new system module of application in jLIVE Builder. You will get following display when click the " Create New Application " button.

¡@

  • Prj. Name : Name of the application must be unique and can not be null.

  • Http : You have to key in IP/Host name and port. The following example use " IP " and " 8080 " port. The string " /servlet/com.probe.jslive.ProbeServlet " is not allow to edit otherwise application will not execute. Check the " Compress Transmit data " checkbox, in order to improve the transmitting speed.

  • Login : Check the " Prompt User Login Dialog " checkbox.  Your project will show login dialog in the beginning. To login the application, you need ID and Password.

  • Author : Author name. Allow null.

  • Version : Version of the application. Allow null.

  • Path : The directory you choose to deployment the application resource.

  • Access Key : This is for workgroup used. It is your access key. It will be necessary when other team member scan your program.

  • Workgroup : The first field is to stand for IP/name and second field is to represent  " AccessKey ". You can set unlimited group member and each one have a pair of " IP ", " Access Key " .

  • Description : A memo for the application.
