8.3.1 DS

  • Datasource : Choose one datasource.

  • Datafield : Choose one field inside datasource.

  • Editable : If you check it, you can edit the string. Otherwise It will only display.

  • Select All : When your mouse click into edit box, it will mark all string. Otherwise don't.

  • Input Method : Use fixed input method.

  • Focus Accelerator : Prompt focus at the component by push " ALT " button and your defined key word at the same time.

  • Use formula : It can result contents by use formula.

  • Effect Initial only : Only display the string at beginning then retain the status from that time.

  • Alignment : Your input string begin position.

8.3.2 Validate

  • Use default validate : It will transform the data type into string.

  • Data Type : You can pick a type for yourself.

  • Read format pattern : You can choose one pattern show with read mode.

  • Edit format pattern :  You can choose one pattern show with edit mode.

  • Constraint ? character length : You can limit the input length.

8.3.3 Event

¡@

The component can connect six kinds of way to fire up event.

  • Action : When you click " Enter " button.

  • Change : When input content change.

  • Gain Focus : While the mouse click into the edit box.

  • Lost Focus : While the mouse click into other place.

  • Double Click : While double click the right button of mouse.

  • Right Click : While click right button of mouse.
