アプリケーション(Application)

jLIVE アプリケーションはデータベースフォーム、リポート及び実行プログラムで構成されている。 新規作成ボタンを押すと、コントロールパネル中にアプリケーションモジュールと各ノードツリーが表れる。アプリケーションを新規作成後、jLIVE  ミドルウェアをオープンし、所在のURLを指定し、テストボタンを押して、接続テストを行ってください。
アプリケーション新規作成 :  新しいアプリケーションを作成する。
ミドルウェア接続テスト :  ミドルウェアが起動されているかどうかをテストする、同時に、URLの設定が正確かどうかを検証する。
キャンセル :  キャンセル
保存 :  変更を保存する。
削除 :  アプリケーションを削除する。システムはコントロールパネル中のノードツリーを消すだけ、関連するデータファイルは削除されない。
プロジェクト名 :  jLIVE  一つのプロジェクトの中、幾つかの実行方式異なるプログラムを生成することができる。
Http :  jLIVE  ミドルウェアの URL
ログイン :  ユーザがログインするとき、パスワードを認証するウィンドウをオープンするかどうか。この機能を使用するとき、プログラム中のセキュリティコントロール機能と一緒に使いべし。
作成者 :  作成者
バージョン :  バージョン
パース :  プロジェクトのデフォルトパース 
アクセス　キー :  ワークグループからローカルリソースをアクセスする時の認証コード。任意な文字列でよい
ワークグループ :  jLIVE システムを購入したライセンス数によって、第一コラムにワークグループのコンピュータ名あるいはIPアドレスを記入、第二コラムに該当コンピュータのアクセスキーを記入する。ワークグループ走査を実行すれば、同一プロジェクト名のモジュールを共用することができる。
記述 :  アプリケーション関連情報を記入する。
