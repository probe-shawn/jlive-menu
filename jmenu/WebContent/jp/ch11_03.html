11.2 データソースイベント (data source event)

共用プロパティ (common properties)

データソースナビゲーター (data source navigator)

データソースフィールド値の設定 (set data source record field data)

トリガーデータソース (trigger data source)

クエリーデータソース (query data source)

NON_SELECT Sql コマンド実行 (execute NON_SELECT SQL commands)

ストアドプロシージャの実行 (execute stored procedure)

データベース Transaction Isolation Level の設定(set database transaction isolation level)

データベース Transaction AutoCommit の設定(set database transaction auto commit)

データベース Transaction Commit or Rollback の設定(set database transaction commit or rollback)

　

　

データソースイベント (data source event) はランタイム (runtime)に実行されます：

  • データソースアクセス(ナビゲーター、フィールド、トリガー、クエリー)。

  •  SQL コマンド実行 (SQL、ストアドプロシージャ)。

  • データベース設定(Isolation level、Transaction)
