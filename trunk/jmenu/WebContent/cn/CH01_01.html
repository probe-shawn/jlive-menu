Thank you for using this release of the jLIVE Builder.The jLIVE is a development environment for building applications, applets, and components that can be deployed on any platform.

The jLIVE Builder includes tools useful for developing programs, written by the Java programming language and running on any platform. You can run your application which developed by jLIVE Builder, on browser via internet or run as a standalone java executable jar file. 

jLIVE Builder comes with Probe middleware working under The Tomcat 4.0 Servlet/JSP container. You can use middleware via localhost to interactive with database server, while designing time. For more information about the Probe middleware product, see the section titled "Probe Application Server" and jLIVE Builder "N-Tieres Middleware" in Probe Web -- technology materials.
