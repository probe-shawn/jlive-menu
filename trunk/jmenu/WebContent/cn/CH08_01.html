When you want to design form, you need click the " Design form " button. Then, you can put any component on the form. To set component common properties only need double click the target component.

¡@

8.1.1 Format

  • Component name : Identify one component. Not allow duplicate.

  • Component size : Identify left, top, width, height.

  • Component tool tips : It will show the string that you typed when you move your mouse on the component.

  • Opaque : Component will not be limpid.

  • Visible : Be able to see the component.

  • Append to menu bar : Move component on menu bar.

8.1.2 Border

  • Default border : Default setting is none.

  • Use title : You can choose position and color. It will appear the string on the component.

8.1.3 Font

  • Font name : Choose one kind of font in your computer.

  • Size : Choose the font size.

  • Style : Check one style. Bold and italic.

  • Color : Choose foreground color and background color.
