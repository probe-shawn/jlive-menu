1.2 Feature

jLIVE Builder is a brand new concept for system developers to accelerate creating a robust business application run on any platform via internet. 

  • The total solution for developing application

The RAD (Rapid Application Development) environment with fully visual framework, make the computing technologies easily accessible for productive developer use. High-performance java swing components with MVC (Model, View, controller) internal data-aware architecture, has proven to be highly capable for helping developers adapt to and exploit advances in Event-Driven technology.
The easy and flexible deployment for running your application on browser or standalone executable jarfile.

  • N-tieres architecture, pure java middleware

Suitable for any web server which supports JVM, not restricted in MS Windows NT environment. Application server (middleware) service all kinds of client application and with cascade loading balance to make significant improvements the throughput of data flow. 

  • Distributing computing with just in time formula interpreter

jLIVE Builder includes both web computing and client computing technologies. The run time client site module (browser or workstation) embedded a formula interpret that can just in time compute the user manipulations and response immediately. Some like verify user's input , counting data relative with database fields content and user's input, and the chart data generated, will perform in client site. That will result in high response performance and avoid web server collapse due to massive computing services.

  • The real time user interactive database operation

jLIVE Builder process the user SQL request that invoke by visible components and communicate with middleware to obtain the result. The middleware connect and process all the database operations by JDBC driver, retrieve result sets then sent back to the requested client.

  • The Event-Driven, object-oriental programming

Easy deploy, no matter what web database usage, The developer can focus all efforts on domain know-how implements to create robust business applications.

  • Probe report system

jLIVE report system is highly integrated with FORM design frame work , it support the same visual RAD environment as FORM design, drag and drop printable object to layout report. The printable content is dynamic computing by jLIVE formula, so all the components content and data source can be printed. The report data source is dynamically provided by application server, and computing by client site. this distributed computing mechanism will make the report system more robust and efficiency.
