【Northwind】-- Sample Application

　

Order Report

　

報表名稱 (Report Name)：OrderReport
　 　
操作說明 (Specifications)：
　

  • 列印訂單資料。
報表資料來源 (Data Source)：
OrderDetails：

訂單資料副檔
【SQL】：

   Select *

   from   orderdetails

   where  orderID={Orders:orderID}
報表元件 (Report Components)：
通用報表元件

元件內容：產品名稱 (Product Name) 。
列印運算式：Condition((Seek("Products","ProductID", {OrderDetails:ProductID})>0),{Products:ProductName},"")，目的在於把 OrderDetails 欄位 ProductID 內含值以產品名稱替換。運算式的意義是，如果在 Products 的 ProductID 找得到同樣的產品編號，就列印該筆記錄的產品名稱， 否則列印空字串。
通用報表元件

元件內容：單筆項目金額小計。
列印運算式：{OrderDetails:UnitPrice}*{OrderDetails:Quantity}*(1-{OrderDetails:Discount})， 單價*數量*(1-折扣)。
通用報表元件

元件內容：訂單金額小計 (Subtotal)。
列印限制：{OrderDetails:Eof()}，OrderDetails 的資料全部印完之後 (到尾筆)，此元件才會列印。
列印運算式：{subtotal}，Form (ORDR_INFO) 上面的 "subtotal" 元件內含值。
