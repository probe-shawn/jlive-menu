字串的運算式，處理字串資料的各種運算式。運算時直譯器 (interpreter)，會將字串運算的參數，轉型 (cast) 成字串型別，再行運算。

　

x $ y

　

說明： x 字串加上 y 字串。"$" 兩邊都必需有值，不得空白。
參數： "x"、"y"，可為常數值或運算式。
傳回值： 相加後的字串。
範例：

假設系統日期為 "2000/11/10"
"ABC" $ "DEF"
" AB" $ "DE"
"ABC " $ 123
" ABC " $123
"ABC" $ -123
" -123 " $ Today()

LTrim(x)

　

說明： 將 x 去左邊空白。
參數： "x"，可為常數值或運算式。
傳回值： 去左空白後的字串。
範例：

LTrim(" ABC   " )             傳回         "ABC   " 
LTrim("A  BC" )               傳回         "A   BC"。

Find(x,y,pos)

　

說明： 在 x 中找尋字串 y，從 x 的第 (pos) 位找起。
參數： "x"、"y"、"pos"，可為常數值或運算式。

範例：

Token(str, sep, nth)

　

說明： 傳回 str  字串中以 sep 分隔的第 nth 個資料。
參數： str ：含分隔符號的字串。
sep：分隔符號。
nth：第 N 個，以 1 起算。
傳回值： 字串。
前後空白皆屬有效資料。
sep 為 空字串傳回原字串 str。
以下情況傳回空字串：
nth 不為數值。
nth < 1 或 nth > 最大資料個數。
範例：

Token("abcd; 歡迎光臨; 1234", ";", 2 )          傳回      "歡迎光臨"
Token("abcd; 歡迎光臨; 1234", ";", 4 )          傳回      ""
Token(";;歡迎光臨; 1234", ";", 3)                   傳回      "歡迎光臨"
Token(";;歡迎光臨; 1234", ";", 2)                   傳回      ""

　

▲Top

　

TokenCount(str, sep)

　

說明： 傳回 str  字串中以 sep 分隔的資料有幾個。
參數： str ：含分隔符號的字串。
sep：分隔符號。
傳回值： 數值字串。
範例：

TokenCount("abcd;歡迎光臨;1234" )          傳回       3
TokenCount("abcd;歡迎光臨;1234;" )         傳回       3
TokenCount("abcd;歡迎光臨;1234; " )         傳回       4 (最後的; 前含一空白)

TokenCount(";;abcd;歡迎光臨;1234" )         傳回       5

TokenCount("abcd;歡迎光臨;1234" )          傳回       2
TokenCount("abcd;歡迎光臨;1234;" )         傳回       3
TokenCount("abcd;歡迎光臨;1234; " )         傳回      3 (最後的; 前含一空白)

TokenCount(";;abcd;歡迎光臨;1234" )         傳回      4
