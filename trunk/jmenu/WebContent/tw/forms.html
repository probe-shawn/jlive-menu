視窗
建立一個新的工作視窗。視窗編輯環境，以完全視覺化拖曳方式設定，設計者可以隨時設計，馬上執行，測試執行狀態，迅速更正錯誤。
新增 :  新增視窗。
取消 :  取消新增。
存檔 :  存檔。
刪除 :  刪除。
製作 Java 程式 :  由 XML 檔產生 Java 的原始程式。
查看 Java 程式 :  查看所產生的 Java 原始程式。
編譯 Java 程式 :  編譯 compile Java 程式。
製作 編譯 Java class :  重新從 XML 產生 Java 原始程式並編譯成 class。
Class 名稱 :  登錄工作視窗名稱，視窗名稱不可填空白。必須以英文命名且遵循 Java 變數的命名法則，系統將以此名稱產生 Java class 檔，並在應用程式中參照。改變已存在的視窗 Class 名稱系統將檢查是否重複並將相關的程式文件一併予以更名
作者 :  程式設計者名稱。
版本 :  視窗版本別。
類型 :  工作視窗分類，可於控制面板檢視時暫時隱藏。
XML 檔 :  建立或指定一個放置 XML 檔資料夾。第一次進入編輯工作視窗時會產生一個  XML 檔，XML 檔中記錄視窗的所有屬性及程序。
Java 程式碼 :  將由系統產生的 Java 程式碼。
Class 路徑 :  建立或指定一個放置 Java class 的檔案資料夾。
文件附註 :  設計者對於工作視窗相關說明。
