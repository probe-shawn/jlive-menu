報表
建立一個新的報表工作表單，於該工作視窗下。報表設計環境模擬真實紙張大小，報表元件可以直接在絕對位置上放置，且各元件皆為獨立物件，不會互相干擾位置。
新增 :  新增報表。
取消 :  取消新增。
存檔 :  存檔。
刪除 :  刪除。
製作 Java 程式 :  由 XML 檔產生 Java 的原始程式。
查看 Java 程式 :  查看所產生的 Java 原始程式。
編譯 Java 程式 :  編譯 compile Java 程式。
製作 編譯 Java class :  重新從 XML 產生 Java 原始程式並編譯成 class。
Class 名稱 :  登錄工作報表名稱，報表名稱不可填空白。必須以英文命名且遵循 Java 變數的命名法則，系統將以此名稱產生 Java class 檔，並在應用程式中參照。改變已存在的報表 Class 名稱系統將檢查是否重複並將相關的程式文件一併予以更名
作者 :  程式設計者名稱。
版本 :  報表版本別。
XML 檔 :  建立或指定一個放置 XML 檔資料夾。第一次進入編輯工作報表時會產生一個  XML 檔，XML 檔中記錄報表的所有屬性及程序。
Java 程式碼 :  將由系統產生的 Java 程式碼。
Class 路徑 :  建立或指定一個放置 Java class 的檔案資料夾。
文件附註 :  設計者對於工作報表相關說明。
