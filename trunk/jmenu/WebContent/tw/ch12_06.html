Abs(x)

　

說明： x的絕對值。
參數： x (可為常數值或運算式)。
傳回值： x的絕對值。
範例：

Abs(-100)           傳回      100

Round(x)

　

說明： 將 x 四捨五入到個位數。
參數： x (可為常數值或運算式)。
傳回值： 四捨五入後的值。
範例：

Round (23.65)     傳回    24

　

▲Top

Truncate(x)

　

說明： 將 x 無條件捨去小數。
參數： x (可為常數值或運算式)。
傳回值： 捨去小數後的值。
範例：

Truncate(23.65)          傳回        23

　

▲Top

傳回值： 捨去後的值。
範例：

Random(seed)

　

說明： 以"seed" 為亂數種子產生亂數值。
參數： seed (可為常數值或運算式)。
傳回值： 亂數值。
範例：

Random(12345 )             傳回            1553932502

　

▲Top

　

　

Random()

　

說明： 以系統目前時間(毫秒 minisecond)為亂數種子產生亂數值。
參數： 無。
傳回值： 亂數值。
範例：

Random() 。

　

▲Top

　

FormatNumber(pattern, value)

　

說明： 將 value 以 pattern 格示化。
參數： pattern：數值格式化樣版，參閱 數值格式化符號。
value：數值。
傳回值： 格式化的數值。
範例：

FormatNumber("###,###,##0.00", 12345678.345)
傳回 12,345,678.35。
