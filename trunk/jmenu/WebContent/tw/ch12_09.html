DsNavigator(ds,op_str)

　

說明： 功能與導覽元件相同，可執行資料來源的各種操作，如備新增、修改、刪除...等。
參數： "ds" 資料來源名稱，須用引號括起來，"op_str" 資料來源的各種操作，細節如下。
first ： 首筆。當資料在首筆時，按鍵會失效。
 prior ： 上一筆。當資料在首筆時，按鍵會失效。
 next ： 下一筆。當資料在尾筆時，按鍵會失效。
 last ： 尾筆。 當資料在尾筆時，按鍵會失效。

insert ： 新增。在新增狀態或編輯狀態時，按鍵會失效。
 edit ： 編輯。
 delete ： 刪除。按刪除鍵時，不會出現警示訊息，請小心。
 post ： 存檔。
cancel ： 取消新增狀態或編輯狀態。
 refresh ： 更新。
 reopen ： 重新開啟資料來源。
 deleteall ： 將資料全部刪除，只適用於 Buffered 資料來源。
 query ： 查詢。
傳回值： 1 表執行成功，0 表失敗。
範例：

DsNavigator("BASC_CUST" ,"next" ) ，移動客戶資料 ("BASC_CUST" ) 到下一筆記錄 ("next")。

DsMoveto(ds,recno)

　

說明： 移動資料來源的記錄指標。若指定的記錄編號不合理 (例如負數或超出尾筆)，則資料來源不動作，記錄指標停在原始位置。
參數： "ds" 資料來源名稱，須用引號括起來，"recno" 記錄編號，可為常數值或運算式。
傳回值： 1 表執行成功，0 表失敗。
範例：

DsMoveto("BASC_CUST" , 15)，移動客戶資料 ("BASC_CUST" ) 到第15筆記錄。

　

▲Top

　

　

PrevRecord(ds,fld)

　

說明： 資料來源上一筆記錄的欄位值。函數作用不會移動記錄指標。
參數： "ds" 資料來源名稱，"fld" 欄位名稱，須用引號括起來，可為常數值或運算式。
傳回值： 欄位內含值，若已在首筆，則傳回空值。
範例：

PreRecord("BASC_CUST" , "CUST_ID")，傳回上一筆客戶 ("BASC_CUST" ) 的 ID ("CUST_ID")。

參數： "ds" 資料來源名稱，"fld" 欄位名稱，須用引號括起來，可為常數值或運算式。
傳回值： 欄位內含值，若已在尾筆，則傳回空值。
範例：

NextRecord("BASC_CUST" , "CUST_ID")，傳回下一筆客戶 ("BASC_CUST" ) 的 ID ("CUST_ID")。

RecordData(ds,fld,recno)

RecordData(ds, fld, recno, exact)

　

說明： 資料來源第 (recno) 筆記錄的欄位值。函數作用不會移動記錄指標。
參數： "ds" 資料來源名稱。
"fld" 欄位名稱，須用引號括起來。
"recno" 記錄編號，可為常數值或運算式。
"exact" 未給值時，預設為 0。
"exact" ：1 -- 當 recno 記錄編號，不存在時，傳回空字串。
"exact" ：0 -- 當 recno 記錄編號，不存在時，傳回空字串，但是負數 (< 0) 時，傳回最後一筆記錄 fld 欄位值。零 (0) 時，傳回目前記錄fld 欄位值。
傳回值： 欄位內含值，若指定的記錄編號不合理 (例如超出尾筆)，則傳回空值。
範例：

RecordData("BASC_CUST" , "CUST_ID" , 15)，傳回第 15 筆客戶 ("BASC_CUST" ) 的 ID ("CUST_ID")。

範例：

IsDelete("BASC_CUST" , 15)，第 15 筆客戶資料 ("BASC_CUST" ) 是否已被刪除。

Seek(ds,fld,value)

　

說明： 尋找資料來源 ("ds") 欄位名稱為 ("fld") 內含值是 ("value") 的記錄。
若找得到則記錄指標會移動到該筆。
參數： "ds" 資料來源名稱，"fld" 欄位名稱，須用引號括起來，"value" 可為常數值或運算式。
傳回值： 該筆記錄編號。找不到傳回 0 。
範例：

Seek("BASC_CUST" , "CUST_ID" , "CS137")，尋找客戶資料 ("BASC_CUST" ) 的客戶編號 ("CUST_ID") 等於 "CS137" 的記錄。

　

▲Top

　

　

Seek(ds,fld,value,seek_ops)

　

說明： 尋找資料來源 ("ds") 欄位名稱為 ("fld") 內含值是 ("value") 的記錄。
"seek_ops" 是搜尋規則，意義如下：
0 ： 全等
1 ： 起始相同
2 ： 內容中含有
3 ： 結尾相同
參數： "ds" 資料來源名稱，"fld" 欄位名稱，須用引號括起來，"value" 可為常數值或運算式。
傳回值： 該筆記錄編號。找不到傳回 0 。
範例：

Seek("BASC_CUST" , "CUST_ID" , "CS137" , 1)，尋找客戶資料 ("BASC_CUST" ) 的客戶編號 ("CUST_ID") 開頭等於 "CS137" 的記錄。

　

▲Top

　

　

Seek(ds,fld,value,seek_ops,moveto)

　

說明： 尋找資料來源 ("ds") 欄位名稱為 ("fld") 內含值是 ("value") 的記錄。
"seek_ops" 是搜尋規則，其意義參考前項函數。
"moveto" 代表找到該筆資料之後，記錄指標是否移動。
參數： "ds" 資料來源名稱，"fld" 欄位名稱，須用引號括起來，"value" 可為常數值或運算式。
"moveto"=1 表示尋找並移動，0 則不移動。
傳回值： 該筆記錄編號。找不到傳回 0 。
範例：

Seek("BASC_CUST" , "CUST_ID" , "CS137" , 1, 1 )，尋找客戶資料 ("BASC_CUST" ) 的客戶編號 ("CUST_ID") 開頭等於 "CS137" 的記錄，且記錄指標移動到該筆。

　

▲Top

　

　

Seek(ds,fld,value,seek_ops,moveto,from)

　

說明： 尋找資料來源 ("ds") 欄位名稱為 ("fld") 內含值是 ("value") 的記錄。
"seek_ops" 是搜尋規則，其意義參考前項函數。
"moveto" 代表找到該筆資料之後，記錄指標是否移動。
"from" 從第幾筆資料(含該筆)開始尋找。
參數： "ds" 資料來源名稱，"fld" 欄位名稱，須用引號括起來，"value" 可為常數值或運算式。
"moveto"=1 表示尋找並移動，0 則不移動。"from" 從第幾筆資料(含該筆)開始尋找。
傳回值： 該筆記錄編號。找不到傳回 0 。
範例：

Seek("BASC_CUST" , "CUST_ID" , "CS137" , 1, 1 , 100)，從第 100 筆資料開始尋找客戶資料 ("BASC_CUST" ) 的客戶編號 ("CUST_ID") 開頭等於 "CS137" 的記錄，且記錄指標移動到該筆。

SeekNext(ds)

　

說明： 以最後一次執行 Seek(ds, ...) 指令，尋找資料來源 ("ds") 的條件，往下搜尋下一筆符合條件的資料記錄。
參數： "ds" 資料來源名稱。
傳回值： 該筆記錄編號。找不到傳回 0 。尚未執行過 Seek(ds, ...) 亦傳回 0 。
範例：

SeekNext("BASC_CUST")。

GetSpOutParam(spname.paramname)

　

說明： 擷取執行預儲程序 ("apname") 之後，輸出參數 ("paramname") 的內含值。
相關內容請參考 執行預儲程序。
參數： "spname" 預儲程序名稱，"paramname" 輸出參數名稱。
傳回值： 輸出參數的內含值，若無此參數傳回空字串。。
範例：

GetSpOutParam("AUTOENCODE.@RETCODE")，取得執行預儲程序 ("AUTOENCODE") 之後，輸出參數 ("@RETCODE") 的內含值。

"@RETCODE" 是 Microsoft SQL 2000 輸出參數的寫法。輸出參數名稱的寫法各資料庫不盡相同，請參考資料庫使用手冊。

GetSqlErrcode(ds.op)

   

說明： 擷取資料來源 ("ds") 在執行資料操作 ("op") 之後，由Probe middleware 或資料庫傳回之錯誤代碼。相關內容請參考 SQL Error Handle。
參數： "ds" 資料來源名稱，"op" 操作內容 (如 insert 、post...etc)。
傳回值： SQL 錯誤代碼。若無此項錯誤傳回 0。
範例：

GetSqlErrcode("Customer.insert")，檢查客戶資料 ("Customer" ) 執行新增動作的結果，一般而言，0 表執行成功。

範例：

SwapRecord("BUFF_TBL" , 2 , 5 )，將資料來源 ("BUFF_TBL" ) 第 2筆記錄與第5筆對調。

InsBufdsd(bufds, rec_data, fld_sep)
InsBufdsd(bufds, rec_data, fld_sep,rec_no)
InsBufdsd(bufds, rec_data, fld_sep,rec_no,moveto)

　

說明： 在 Buffer 資料來源 ("bufds")的 (rec_no) 處, 插入 rec_data, 其中 rec_data 資料中的欄位以 fld_sep 分隔。
參數： bufds：資料來源名稱。
rec_data：資料記錄。
fld_sep：rec_data 中欄位 (field) 的分隔字串。
rec_no：記錄編號，可為常數值或運算式。0及大於目前記錄數將會插入(insert) 至最後
moveto：在插入資料後, 將目前記錄指標移至該記錄。
傳回值： 1 表執行成功，0 表失敗。
範例：

將資料記錄 "abcd\t1234\tdefg" 加為資料來源 ("BUFF_TBL")最後一筆, 並將記錄指標移至該筆。
InsBufdsFld("BUFF_TBL" ,"abcd\t1234\tdefg" , Char(9),  0 , 1)。

BulkInsBufds(bufds, rec_data, fld_sep,rec_sep)

　

說明： 將大量多筆資料紀錄 (rec_data) 加到Buffer 資料來源 ("bufds")的最後(append), rec_data 資料中以 rec_sep 分隔每筆記錄, 每筆記錄中以fld_sep 分隔欄位值。
參數： bufds：資料來源名稱。
rec_data：多筆資料紀錄值。
fld_sep：欄位 (field) 的分隔字串。
rec_sep：記錄 (record) 的分隔字串。
傳回值： 1 表執行成功，0 表失敗。
範例：

將 3 筆資料記錄 "aaaa\t1111\t2222\nbbbb\t3333\t4444\ncccc\t5555\t6666" 附加到資料來源 ("BUFF_TBL")。
BulkInsBufds("BUFF_TBL" , "aaaa\t1111\t2222\nbbbb\t3333\t4444\ncccc\t5555\t6666" , Char(9), Char(13)$char(10))。
