FileChooser(title,isfile,filter)

　

說明： 開啟目錄或檔案的選擇視窗。
參數： "title" ：視窗標題。
"isfile" ：1 表限制只能選擇檔案，0 表限制只能選擇目錄。
"filter"：篩選副檔名。0 表所有檔案，".txt" 表限定副檔名為 ".txt" 的檔案。
傳回值： 客戶端 (client) 所選擇的目錄或檔案的路徑 (path)，若客戶端 (client) 按 "取消"按鈕，則傳回空值。
範例：

FileChooser("Choose Files" ,1 ,".txt" ) 。

FileChooser(title, isfile, filter, startpath)

　

說明： 開啟目錄或檔案的選擇視窗。
參數： "title" ：視窗標題。
"isfile" ：1 表限制只能選擇檔案，0 表限制只能選擇目錄。
"filter"：篩選副檔名。0 表所有檔案，".txt" 表限定副檔名為 ".txt" 的檔案。
"startpath"：從 startpath 的目錄開啟。
傳回值： 客戶端 (client) 所選擇的目錄或檔案的路徑 (path)，若客戶端 (client) 按 "取消"按鈕，則傳回空值。
範例：

FileChooser("Choose Files" ,1 ,".txt", "c:\jlive\" ) 。

CopyTo(source,target)

　

說明： 將檔案複製到所指定的位置，指定方式是以UNC路徑加上檔名。
ps: UNC 為 Universal Naming Convention 的縮寫。
參數： "source" ：檔案來源路徑檔名。( \\server\path\檔案名稱 )
"target" ：目標路徑檔名。( \\server\path\檔案名稱 )
傳回值： 1 表執行成功，0 表失敗。
範例：

CopyTo( "\\localhost\sample\test1.doc" , "\\server-3\williams\tes2.doc")。

Load(source)

　

說明： 將檔案以字串型態讀入，指定方式是以UNC路徑加上檔名。
參數： "source" ：檔案來源路徑檔名。( \\server\path\檔案名稱 )
傳回值： 成功讀取則傳回檔案內容資料，否則傳回空字串。
範例：

Load( "\\server-3\williams\tes2.html")。

　

▲Top

　

Load(source, charset)

　

說明： 將檔案以字串型態讀入，指定方式是以UNC路徑加上檔名。讀入時以 charset 編碼
參數： "source" ：檔案來源路徑檔名。( \\server\path\檔案名稱 )
"charset" ：編碼字集，例如："MS950"、"MS932"。
傳回值： 成功讀取則傳回檔案內容資料，否則傳回空字串。
範例：

Load( "\\server-3\williams\test2.html", "MS950")。

File(cmd,target)

　

說明： 執行目錄與檔案的管理功能。
參數： "cmd" ：命令，分為以下五種。
      md：建立目錄， File( "md" , "\\server\sharePath\目錄" )。
      rd：移除目錄， File( "rd" , "\\server\sharePath\目錄" )。
      del：刪除檔案， File( "del" , "\\server\sharePath\檔名" )。
      exist：檢查目錄或檔案是否存在， File( "exist" , "\\server\sharePath\目錄或檔名" )。
      deltree：刪除目錄、子目錄、及所有檔案， File( "deltree" , "\\server\sharePath\目錄名" )。
"target" ：目標目錄或檔名路徑。( \\server\sharepath\檔案名稱 )

SaveAs(target, data)

　

說明： 將 data 字串存入 target，target 指定方式是以UNC路徑加上檔名。
參數： "target" ：目的檔案路徑檔名。( \\server\path\檔案名稱 )
data ：字串或運算式結果值。
傳回值： 1 表執行成功， 0  表失敗。
範例：

SaveAs( "\\server-3\williams\test.txt", {memo})。

　

▲Top

　

SaveAs(target, data, append)

　

說明： 以附加於檔尾方式將 data 字串存入 target，target 指定方式是以UNC路徑加上檔名。
參數： "target" ：目的檔案路徑檔名。( \\server\path\檔案名稱 )
data ：字串或運算式結果值。
append：1 表附加，0  表覆蓋原資料。 
傳回值： 1 表執行成功， 0  表失敗。
範例：

SaveAs( "\\server-3\williams\test.txt", {memo}, 1)。

FTP(host, port(21), id, pass, CMD, workdir, remote, local)

　

說明： 執行 FTP (File Transfer Protocol) 作業。
參數： host：FTP 主機名稱。
port(21)：FTP port，一般為 21 。
id：登入FTP 主機的 ID。
pass：登入FTP 主機 PASSWORD。
CMD：執行 FTP 的命令。RETR -- 讀取檔案；STOR -- 上傳檔案。
workdir：登入FTP 主機並切換至目錄 workdir。
remote：FTP 主機檔案名稱。
local：使用者端檔案名稱
傳回值： 1 表執行成功， 0  表失敗。
範例：

讀取 server-2 主機上，目錄 shared_doc 中的 customer_list.txt 檔案，存入為本機檔案名稱 c:\ftp_doc\customer.txt：

FTP("server-2", "21", "probe", "probe", "RETR", "shared_doc", "customer_list.txt", "c:\ftp_doc\customer.txt")

上傳本機 "c:\ftp_doc\summary.txt 至 server-2 主機，目錄 shared_doc 中，並更名為 summary_今天日期.txt

FTP("server-2", "21", "probe", "probe", "STOR", "shared_doc", "summary_"$(Today()[1 to 8])$".txt", "c:\ftp_doc\summary.txt")
