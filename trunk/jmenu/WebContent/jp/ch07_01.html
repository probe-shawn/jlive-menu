7.2 レポート管理 (report arrangement)

　

jLIVE Builder™ パネルの中，レポートシステムはフォーム(　form　)下の節(　node　)に装備されています｡デザインナーはニーズによって多数の機能の異なるレポートを追加できます｡既に存在しているレポートに対し，ページ編集、保存、削除、コンパイルをすることができます｡

　

｡

  • デザイン (design)  デザインモードへ｡

  • Java™ (make Java™)   Java™ ソースコード作成｡

  • Java™ (compile Java™)コンパイル 　Java™ ソースコードコンパイル｡

  • ビルドJava™ Class (build Java™ class) 　Java™ ソースコードを class　にコンパイル｡
