リポート(Report)
新しいリポートを作成する。リポートはフォームと同じ手法で設計することができる。プリントコンポーネントを直接紙上の絶対位置に置くことができる。
リポート新規作成 :  デザインのため、新しいリポートを作成する。
キャンセル :  新規作成を取り消す。
保存 :  変更を保存する。
削除 :  リポートを削除する。
Java生成 :  XMLファイルからJava ソースを生成する。
Javaビュー :  Java ソースを見る。
Javaコンパイル :  Java ソースをコンパイルし、クラスを生成する。
構築 :  XMLファイルからJava ソースを生成し、コンパイルかけ、クラスを生成する。
クラス名 :  リポートの名前を登録する。リポートの名前は空白にしてはならない。英文字のみしか使えない且つJava の変数命名法則に従ってください。システムははこの名前からJava クラスを作成し、アプリケーションから参照する。既存のリポート名(クラス名)を変更すると、システムはまず名前が重複しているかどうかをチェックし、関連しているプログラムやファイル名を共に変更する。
作成者 :  作成者。
バージョン :  バージョン。
カテゴリ :  リポートを分類し、コントロールパネルで表示させたり、させなかったりすることができる。
Javaソース :  システムからJava ソースを生成されます。
クラスパース :  Java クラスを保存するパースを指定する、あるいは新規作成する。
記述 :  相関情報をプログラマーが記入しま
