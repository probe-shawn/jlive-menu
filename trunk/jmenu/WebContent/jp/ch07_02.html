7.3 レポートプロパティ (report properties)

Java™ 印刷セキュリティ体制(Java™ print security policy)

　

jLIVE Builder™ レポートシステムはレポートプロパティによって設定します (report properties)，デザイナーはニーズによって印刷作業の設定を行えます。例えば：

  • ユーザにプレビューが必要かどうか(preview)。

  • ユーザにプリンター設定が必要かどうか(printer setup)。

  • ユーザにデフォルト印刷部数設定が必要かどうか(copies)。

  • レポート種別によってプリンター指定するかどうか(printer)。

レポートプロパティは編集 ( レポートプロパティ編集 ) をクリックし、次のフォームが表示されます。

　

  • 印刷作業 (print job)：

    　

    • 印刷作業名 (print job name)：印刷作業名称としてPrint Job query に表示されます。

      　

    • データソース (data source)：データソースの印刷(data source)。

      　

    • プレビュー (preview window)：

    • プレビューウインドウ表示 (show preview windows)：選択すると、印刷する時にプレビューウインドウが表示されます(preview window)。ユーザは印刷するかどうかを決めます。

      　

    • プリンターセットアップ (show printer setup dialog) ：選択すると、プリンター設定ページが表示され、設定値の変更ができます。

    • コピー (copies) ：コピー部数。

      　

    • ヘッダ、フッタ?Em (header & footer location)：

      　

      • ヘッダドット(点) (header dot)：ヘッダ分割線位置，単位はピクセル。

      • フッタドット(点) (footer dot)：フッタ分割線位置，単位はピクセル。

      • ヘッダ(インチ) (header inch)：ヘッダ分割線位置，単位はインチ。

      • フッタ(インチ) (footer inch)：フッタ分割線位置，単位はインチ。

      　

    • ページ Media タイプ (page media type )：スタンダード寸法オプション。例えば iso、jis など， 73種類サポート。

    　

　

  • ページカラー (page color type)：カラー印刷かモノクロー印刷の設定。

    　

    • カラー (color)

    • モノクロー (monochrome)

    　

  • 印刷向き (page orientation requested type)：垂直か横に印刷。

    　

  • ページ印刷品質(page print quality type)：品質プション。

    　

    •  (high)

    •  (normal)

    •  (draft)

      　

  • プリンター (printer)：プリンター指定。

    　

    • デフォルトプリンター (default printer) ：デフォールトプリンター変更可能。

      　

　

  • デフォールトでないプリンター：フォーミュラー(formula)でプリンターを選びます。stringかformulaでダイナミック的（dynamic)にプリンター名を指定します。リモートプリンターの場合，ネットワークプリンター名を指定します。

    以下のようになります：

    　

    \\プリンタサーバー名\ネットワークプリンター名
