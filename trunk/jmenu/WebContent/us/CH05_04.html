5.4 Design Form

When you want design form, you can select the form and click the mouse right button. Then choose " Design " or click the button which was marked with a red circle as follow images.

After you enter the design panel, you will see four main function on the top. 

  • System

    • Back to Console : Exit design mode and back to main panel.

    • Save Modified : Save the changes. 在設計模式下, 所有的變更都尚未儲存, 因此當程式有類似迴圈的邏輯時, 建議在測試執行前先行存檔動作, 以免進入無窮迴圈後程式必須強制中止而無法存檔, 而喪失存檔前的所有設計內容

    • Reload Images : When you add, delete or alter those images in preloadimages and images directories, you have to reload images again. Then the new images will display in the select option.

  • View Java Source : Open the java source code. See the example picture.

  • jLIVE  Builder 提供程式設計師自行加入 java source code 加入的方法為利用 xml 的標籤  <reserve> </reserve> 將程式內容用此標籤包住, 然後放到執行所需要的位置再 compile 成為 java source code ( *.java ) 這時候 你會發現加入的程式碼會被放在 source code 的最下面, 你必須要把這一段新加入的程式碼剪下放回你要的地方, 然後再將 *.java  compile 成 class.

  • Hash & SysHash Editor : This is only effective in design mode. When the form need hash or syshash parameter in initial state, you can add key and value.

　

  • Forms

    • Form Properties : Open the form properties or click the right button as follow image.

      　

      　

    • Run : Test run the form. Or click the left button.

    • Design : When you want close test run, you can click this button. Or click the middle button.

　

  • Edit

    • Copy :  Copy the selected components.

    • Paste :  Paste the components that copied earlier.

    • Delete : Delete the selected components.

　

  • Alignment : You need to select components first.

    • Undo

    • Alignment to form left

    • Alignment to form top

    • Alignment to form right

    • Alignment to form bottom

    • Horizontal left

    • Horizontal center

    • Horizontal right

    • Horizontal form center

    • Horizontal equal space

    • Vertical top

    • Vertical center

    • Vertical bottom

    • Vertical form center

    • Vertical equal space

    • Delete

　

After these action, you will see the design mode window. You can drag the components on tool bar and drop to the design panel. If you want use one component, you have to click the component and move mouse to the position you want. Then click the mouse left button. It will be dropped  there. You can move it to another place or change its size. Only move the blue point.

 

當選取範圍包含多個元件時, 控點會成標示為綠色, 這時你可以一次移動多個元件或改變字型及元件大小

要一次改變所有元件大小和字型的方法是先選取目標, 然後在上方點滑鼠右鍵即可出現下列子視窗, 請點選後進入修改

當元件在 tab 中時, 標示的控點為粉紅色, 這時並不能一次選取多個元件, 只能選取一個
