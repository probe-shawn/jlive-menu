7.2 Jar Run

Jar run could execute standalone. It's no more need to load any class from server because all class include in the jar file.  And it can not update class from server, this is browser only.

  • App Name : Application name.

  • MainClass Name : The initial class when application execute.

  • Application Title : Application title.

  • Resizeable Frame : You can change main form size if you check resizable check box.

  • Initial State : When application execute, the initial form size have three modes.

    • Normal : Follow main form original size

    • Iconfied : Icon size

    • Maximize : full screen

  • Main Panel : You can choose one form to be main panel.

  • Custom JarFile : You can use any other jar file to increase capability.
