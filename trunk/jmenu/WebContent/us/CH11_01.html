11. DataSource

Content:

Table list ~ Properties

DataSource

¡@

Data Source -- The data that will be manipulated within the form.

Data source type

  • Table data source -- result set of simple SQL select table command, updatable.

  • SQL data source -- result set of complex SQL join tables command, readonly.

  • Developer defined Buffer data source -- table structure which defined by developer for client use

Data source properties

  • Read only, Row fetch, Blob fetch, Globe data source

  • SQL edit pane and parameters

  • Row id fields

  • Default data

  • Operation validate

  • Trigger event

  • SQL error handle
